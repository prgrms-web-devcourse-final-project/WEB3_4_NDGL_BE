데모 이미지 업로드 테스트용

이 페이지는 AWS S3 이미지 업로드 기능을 테스트하기 위한 데모 페이지입니다.

현재 구성:

  • 이미지 타입: POST
  • 참조 ID: 999
  • 최대 파일 크기: 5MB
  • 최대 업로드 개수: 10개

이 페이지에서 업로드된 이미지는 실제 S3 버킷에 저장되며, 데이터베이스에 기록됩니다.

* 참고: 실제 서비스에서는 사용자 인증 후 연관된 ID가 자동으로 설정됩니다.
